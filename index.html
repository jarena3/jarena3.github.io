Projects

Smithy A procedural generation framework for Unity

A utility that helps game developers build assets with complex self-assembling behavior directly in the Unity editor. Users place nodes on meshes or in space and define generation rules. Smithy builds trees from these nodes and recursively fills them, creating new content both in the editor and at runtime.
Smithy comes in several versions, including a version designed specifically for asset artists, which allows content creators to bake node trees and package their products with a free lightweight generator. Currently finishing development.

Flat Editor Modern UI tools and styling for Unity Asset projects Open Source

The open-sourcing and expansion of Smithy's UI into a standalone Unity editor UI toolkit. Unity's editor UI is pretty rough, and the tools Asset developers have for creating and styling editor UI are, at best, pretty anemic.
FlatEditor attempts to emulate of Bootstrap's functionality and style, where possible.

Selectionator A powerful and flexible weighted selection engine for Unity

A small C# utility that provides weighted random selections using binary trees. Selectionator can also "sweeten" selections (often found in game development as "magicfind", or similar), and comes with a variety of techniques for appling different varieties of sweetening.

JLS Homework Utility A class and homework scheduling tool for Korean ESL teachers Open Source

A small WinForms utility built for Jeongsang Language Academy (정상어학원) of Seoul, South Korea. It automates most of the Native Teachers' beginning-of-semester preparation work. It uses LinqToExcel to pull student information from academy branch networks, and Cathal Coffey's DOCX library to write printable, shareable Microsoft Word files. It produces homework schedules for students and parents, class schedules for co-teachers, and class lists and homework tracking documents for teachers.
JLS Homework Utility was built to support both modern and older (XP SP1, primarily) versions of windows.

Trixelator makes cell array overlays of an uploaded image. It then colors the cells based on point selections or average color values. Currently has support for a few flavors of triangle and hexagon cells. In the future, I'd like to add in some more interesting cells. Voronoi, maybe.

Imgruber Screenshot-to-Imgur pipeline utility Open Source

Imgruber is my attempt to get my screenshot-to-forum-post pipeline as short as possible. It's a taskbar application that uploads a section of the screen to Imgur.com using the Imgur API. The URL of the uploaded image is then copied to the clipboard, in the format of the user's choosing — appropriate for a forum post, StackExchange question, E-mail, messenger app, etc.
Imgruber shortened my screenshot-to-URL time from an average of 75 seconds to an average of 5.5 seconds. A total success in my view, but ultimately I've stopped using it due to the excellent (and open source) XShare.
